# Focura Community                                                                               
                                         
Community space for [Focura](https://focura.io) — an ADHD-focused focus session tool for macOS.  
                                                                                                 
Use Discussions to share feedback, request features, or talk about focus workflows.              
                                                          
## Categories                                                                                    
                                                          
- **General** — anything on your mind                                                            
- **Feature Requests** — ideas for what Focura should do
- **Bug Reports** — something not working as expected                                            
- **Show and Tell** — share how you use Focura in your workflow
